# 使用帐户{#working-with-accounts}

使用单个用户和群组帐户控制报表门户中的访问权限和权限。

某些用户可被指定为管理员，以便他们管理所有帐户。

单个用户帐户和组帐户均指定特定用户或组可以访问的配置文件和选项卡。 主要区别在于，单个帐户允许用户重置密码，而组帐户密码只能由[!DNL Report Portal]管理员重置。
