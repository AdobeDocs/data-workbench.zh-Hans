# 使用帐户{#working-with-accounts}

使用单个用户和用户组帐户控制对报告门户中的访问权限和权限。

某些用户可以指定为管理员，使他们能够管理所有帐户。

单个用户帐户和用户组帐户指定特定用户或用户组可以访问的配置文件和选项卡。 主要区别在于，单个帐户使用户能够重置口令，而组帐户口令只能由管理员重置 [!DNL Report Portal] 。
