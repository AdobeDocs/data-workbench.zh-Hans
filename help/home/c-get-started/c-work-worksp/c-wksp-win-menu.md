# 使用工作区“窗口”菜单{#using-the-workspace-window-menu}

使用该菜单，可以向工作区中添加可用的可视化。

右键单击工作区窗口（工作区背景中的任意位置）。

![](assets/mnu_workspace.png)

>[!NOTE]
>
>由于您的Data Workbench实施可以完全自定义，因此显示的菜单项可能与本节中介绍的内容不同。
