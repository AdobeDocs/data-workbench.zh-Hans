# 重新调整工作区{#refit-a-workspace}

{{eol}}

“重新调整工作区”将可视化的大小调整为您所使用的显示分辨率。

在工作区中，单击 **[!UICONTROL File]** > **[!UICONTROL Refit Workspace]**.

该选项对于快速调整演示文稿的显示非常有帮助。
