# 使用矢量图层{#working-with-vector-layers}

{{eol}}

在Data Workbench中，矢量层在地球上显示静态矢量数据（线条图）。

您可以使用矢量层显示地理区域的轮廓，例如州、国家/地区、岛屿或水体。
