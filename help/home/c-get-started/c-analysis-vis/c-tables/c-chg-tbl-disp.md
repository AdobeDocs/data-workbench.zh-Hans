# 更改表显示{#change-the-table-display}

{{eol}}

有关选择显示类型的信息。

* 右键单击可视化的上边框，然后单击以下选项之一：

   * **[!UICONTROL Display As]** > **[!UICONTROL Table]**

   * **[!UICONTROL Display As]** > **[!UICONTROL Bar Graph]**

   * **[!UICONTROL Display As]** > **[!UICONTROL Both]** （默认）

   ![](assets/mnu_Table_Bar_Display.png)

   此时会在活动的显示模式左侧显示一个 X。
