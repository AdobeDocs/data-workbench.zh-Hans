# 启用系列图例{#enable-a-series-legend}

可以对某个维度中的元素以系列的形式进行颜色编码，以便可以跨工作区中的其他可视化轻松跟踪每个元素。

**启用或禁用系列图例**

* 右键单击要进行颜色编码的维度的元素或标签，然后单击&#x200B;**[!UICONTROL Series Legend]**。

   当系列图例处于活动状态时，会在菜单选项旁边出现一个 X 并且维度中的元素会以交替颜色显示。
