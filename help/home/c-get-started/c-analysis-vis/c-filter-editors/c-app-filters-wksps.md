# 将过滤器应用于工作区{#apply-a-filter-to-a-workspace}

对工作区应用过滤器的步骤。

* In the filter editor window, click **[!UICONTROL Apply Filter]**.

   过滤器编辑器窗口的边框变为黄色，表示该过滤器已应用于工作区的某个选择。
