# 图例{#legends}

{{eol}}

图例提供有关工作区中显示的可视化的说明性详细信息，从而使您能够更加详细地探究可视化中表示的数据。

例如，颜色图例可用于通过为可视化提供颜色图来评估性能指标（如转化率或维系率）。同样，量度图例根据您在工作区中进行的选择（查询）显示标准量度的值。
