# 区段{#segments}

区段可用于通过选择动态构建维度。

当使用杂乱、非重叠组（要将该组用作维度）时，区段最有用，例如当基于其之前的购买来比较两种类型的顾客时。

使用区段只影响您在其中使用该区段维度的工作区。
