# 显示量度信息{#display-metric-information}

有关在路径浏览器中显示量度信息的信息。

右键单击某个元素名称可查看位于菜单顶部部分的路径浏览器关联量度的数量和百分比。

例如，如果您使用网站数据，请右键单击某个页面名称，以查看位于菜单顶部的“页面查看次数”的数量和百分比。
