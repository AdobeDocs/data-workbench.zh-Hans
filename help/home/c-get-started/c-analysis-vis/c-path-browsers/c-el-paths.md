# 查看路径中的更多元素{#view-more-elements-in-paths}

{{eol}}

可以选择查看路径中的更多元素。

多数情况下，您的路径浏览器不足够大，无法显示序列中的所有元素。

* 单击 **[!UICONTROL More]** 可视化左下角以查看路径中的先前元素。
* 单击 **[!UICONTROL More]** 视图右下角以查看路径中的后续元素。

通过单击位于可视化左下角或右下角部分的“更多”，打开一个列出路径中先前元素或后续元素的表格。该表格还显示每个元素的关联量度的值。
