# 批注{#annotations}

批注是用户可添加到工作区以包含描述性评论、图表或动态标题的窗口。

还可以将注释作为标注添加到工作区。 有关使用注释标注的信息，请参 [阅将标注添加到工作区](../../../../home/c-get-started/c-vis/c-call-wkspc.md#concept-212b09e763044d938987b4a9c658adc0)。
