# 复杂过滤器描述{#complex-filter-description}

{{eol}}

您可以为“因变量”和“训练过滤器”添加“显示复杂过滤器说明”。

至 **[!UICONTROL Show the Complex Filter Description]**，从评分对话框中选择 [!DNL Options > Show Complex Filter Description].
