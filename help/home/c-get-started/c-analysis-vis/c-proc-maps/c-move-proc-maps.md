# 移动和旋转流程图{#move-and-rotate-a-process-map}

有关在窗口中移动流程图和如何旋转 3D 流程图的信息。

**在窗口中移动流程图**

* 单击并将流程图拖动到所需的位置。

**旋转 3D 流程图**

* 右键单击并拖动流程图，同时向所需的旋转方向移动鼠标。
