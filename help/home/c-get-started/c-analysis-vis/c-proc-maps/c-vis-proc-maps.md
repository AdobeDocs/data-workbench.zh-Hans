# 从流程图创建可视化{#create-a-visualization-from-a-process-map}

您可以从流程图创建其他可视化。

例如，可以创建图表、表格或散点图。

* 右键单击任意节点，然后单 **[!UICONTROL Add Visualization]** 击> *&lt;**[!UICONTROL visualization type]**>*。

您的流程图上的每个节点会成为“网站地图”维度的一个元素。您可以使用所选可视化的功能来分析节点数据。
