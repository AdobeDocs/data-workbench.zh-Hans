# 添加标注{#add-a-callout}

有关向 2D 流程图中添加标注的信息。

右键单击节点，然后单击&#x200B;**[!UICONTROL Add Callout]** > *&lt;**[!UICONTROL callout type]**>*。

有关标注的更多信息，请参阅[将标注添加到工作区](../../../../home/c-get-started/c-vis/c-call-wkspc.md#concept-212b09e763044d938987b4a9c658adc0)。

>[!NOTE]
>
>只能将标注添加到2D流程图。
