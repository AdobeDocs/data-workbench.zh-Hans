## Data Workbench客户端{#section-dfea35452c2e4f0b92229a4974bc24fa}的操作

在客户端用户界面内执行任务时使用这些操作。

| 操作 | 结果 |
|---|---|
| 单击  | 选择项目 |
| 单击并拖动 | 选择范围 |
| 按 Ctrl 并选择 | 添加到选区 |
| 按 Shift 并选择 | 从选区中删除 |
