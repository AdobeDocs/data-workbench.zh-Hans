# 自定义配置文件管理器和服务器文件管理器{#customize-the-profile-manager-and-server-file-manager}

{{eol}}

“配置文件管理器”和“服务器文件管理器”显示大量的目录和文件。

在某些情况下，您可能希望不用导航完整的目录结构便能访问其中某个管理器的子目录。为此，可以创建一个新管理器，仅显示您所选的目录。
