# 处理图例{#processing-legend}

“处理图例”提供有关特定服务器的数据处理和转换的详细信息，可让您跟踪正在进行重新处理和重新转换的数据的进度。

![](assets/vis_ProcessingLegend.png)

  </tr> 
  <tr>
