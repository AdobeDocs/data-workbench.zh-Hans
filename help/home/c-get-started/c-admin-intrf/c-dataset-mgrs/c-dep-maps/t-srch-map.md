# Search within a map{#search-within-a-map}

搜索包含特定文本的组件的步骤。

1. Right-click within the dependency map and click **[!UICONTROL Search]**.
1. 在文本框内，键入要搜索的文本。

   ![步骤信息](assets/vis_DependencyMap_Search.png)

图中将突出显示名称或参数设置包含搜索文本的组件的节点。
