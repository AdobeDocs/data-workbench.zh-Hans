# 显示性能数据{#display-performance-data}

{{eol}}

有关显示转换或扩展维度的性能数据的信息。

* 右键单击依赖关系图，然后单击 **[!UICONTROL Show Performance Data]**. 启用后，在的左侧会显示一个X [!DNL Show Performance Data].右键单击转换或扩展维度节点时，将显示以下性能信息：

   * 执行转换或创建扩展维度所需的 CPU 秒数
   * 存储扩展维度所需的内存量
   * 扩展维度中的元素数量
   * 存储扩展维度中元素名称列表所需的内存量
