# 将文件复制并粘贴为附件{#copy-and-paste-a-file-as-an-attachment}

复制文件并将其作为 Microsoft Outlook 中的电子邮件附件进行粘贴的步骤。

* Right-click the check mark for the file and click **[!UICONTROL Copy]**. 在电子邮件正文中，按 Ctrl+v 附加该文件。
