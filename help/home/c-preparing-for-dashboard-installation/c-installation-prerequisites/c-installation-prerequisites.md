# 安装先决条件{#installation-prerequisites}

在开始安装之前，请使用具有配置服务器和安装新软件权限的管理员帐户登录到Web服务器。

>[!NOTE]
>
>功能板系统可以部署在单层（IIS和数据库位于同一服务器上）中，也可以部署为两层系统。 本指南未介绍两层方法中涉及的其他配置步骤。
