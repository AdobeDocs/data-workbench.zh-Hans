# 安装先决条件{#installation-prerequisites}

在开始安装之前，请使用具有配置服务器和安装新软件权限的用户帐户登录Web服务器。

>[!NOTE]
>
>仪表板系统可以部署在单一层（IIS和同一服务器上的数据库）中，也可以部署为两层系统。 本指南中未涵盖的两层方法中涉及其他配置步骤。
