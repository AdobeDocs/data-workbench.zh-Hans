# 管理用户帐户{#managing-user-accounts}

用户帐户管理是使用管理控制台中的“用户管理器”部分执行的。

可以使用此区域中的工具创建、编辑和删除用户。 您一次可以拥有的用户数取决于获得许可的用户数。

以下所有用户操作均在中的选 **[!UICONTROL User Manager]** 项卡内执行 **[!UICONTROL Administration Console]**。
