# 管理用户帐户{#managing-user-accounts}

{{eol}}

用户帐户管理是使用管理控制台中的“用户管理器”部分执行的。

可以使用此区域中的工具创建、编辑和删除用户。 您一次可以拥有的用户数量取决于获得许可的用户数量。

以下所有用户操作均在 **[!UICONTROL User Manager]** 选项卡 **[!UICONTROL Administration Console]**.
