# 移除用户帐户{#removing-a-user-account}

1. 单击要从系统中删除的用户行末尾的&#x200B;**[!UICONTROL X]**&#x200B;图标。

   系统会显示一个提示，询问您是否要删除该用户。 1.选择&#x200B;**[!UICONTROL Yes]**&#x200B;以删除用户。

   如果操作成功，您将看到一条消息，指示用户已成功删除。
