# 删除用户帐户{#removing-a-user-account}

1. 单击 **[!UICONTROL X]** 要从系统中删除的用户行末尾的图标。

   此时将显示一条提示，询问您是否要删除该用户。 1.选择 **[!UICONTROL Yes]** 以删除用户。

   如果操作成功，您将看到一条消息，指示用户已成功删除。
