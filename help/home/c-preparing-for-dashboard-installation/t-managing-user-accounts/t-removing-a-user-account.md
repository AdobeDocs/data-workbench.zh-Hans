# 移除用户帐户{#removing-a-user-account}

{{eol}}

1. 单击 **[!UICONTROL X]** 图标。

   系统会显示一个提示，询问您是否要删除该用户。 1.选择 **[!UICONTROL Yes]** 删除用户。

   如果操作成功，您将看到一条消息，指示用户已成功删除。
