# 编辑用户帐户{#editing-a-user-account}

1. 单击要编 ![](assets/edit_icon.png) 辑的用户行末尾的齿轮图标

   ![](assets/edit_user_account.png)

1. 编辑用户的帐户设置，然后单击 **[!UICONTROL Save]**。

   如果操作成功，您将看到一条消息，指示用户组已成功保存。
