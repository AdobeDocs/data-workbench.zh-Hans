# 编辑配置文件连接{#editing-a-profile-connection}

{{eol}}

1. 单击齿轮图标 ![](assets/edit_icon.png) 在要编辑的用户档案连接的行末尾。

   ![](assets/edit_profile_connection.png)

1. 编辑配置文件连接的设置，然后单击 **[!UICONTROL Update Connection]** 保存。

   如果操作成功，您将看到一条消息，指示用户档案连接已成功更新。
