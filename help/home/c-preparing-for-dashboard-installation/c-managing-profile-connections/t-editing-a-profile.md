# 编辑配置文件连接{#editing-a-profile-connection}

1. 单击要编 ![](assets/edit_icon.png) 辑的轮廓连接行末尾的齿轮图标。

   ![](assets/edit_profile_connection.png)

1. 编辑配置文件连接的设置，然后单击 **[!UICONTROL Update Connection]** 以保存。

   如果操作成功，您将看到一条消息，指示配置文件连接已成功更新。
