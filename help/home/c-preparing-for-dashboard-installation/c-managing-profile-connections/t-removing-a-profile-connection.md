# 删除配置文件连接{#removing-a-profile-connection}

{{eol}}

1. 单击 **[!UICONTROL X]** 删除与用户档案的连接。
1. 系统将显示一个提示，询问您是否要删除配置文件连接。 选择 **[!UICONTROL Yes]** 删除配置文件连接。

   ![](assets/delete_connection.png)

   如果操作成功，您将看到一条消息，指示已成功删除用户档案连接。
