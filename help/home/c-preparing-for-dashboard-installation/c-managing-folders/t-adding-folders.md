# 添加文件夹{#adding-folders}

1. 右键单击现有文件夹，然后选择&#x200B;**[!UICONTROL New subfolder]**。

   ![](assets/new_subfolder_1.png)

1. 将创建新子文件夹。

   ![](assets/new_subfolder_2.png)

   如果操作成功，您还会看到提示，指示已创建文件夹。
