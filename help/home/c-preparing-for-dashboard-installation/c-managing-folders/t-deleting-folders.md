# 删除文件夹{#deleting-folders}

{{eol}}

1. 右键单击要删除的文件夹并选择 **[!UICONTROL Delete]**.

   ![](assets/delete_folder.png)

   >[!NOTE]
   >
   >文件夹必须为空，然后才能删除。

1. 系统将提示您确保删除文件夹。
1. 如果操作成功，您还将看到一条提示，指示文件夹已被删除。
