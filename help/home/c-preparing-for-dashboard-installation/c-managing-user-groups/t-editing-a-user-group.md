# 编辑用户组{#editing-a-user-group}

1. 单击行末端的齿轮图标以编辑组。

   ![](assets/edit_user_group.png)

1. 编辑用户组的设置，然后单击 **[!UICONTROL Save]** 以保存更改。

   如果操作成功，您将看到一条消息，指示用户组已成功保存。
