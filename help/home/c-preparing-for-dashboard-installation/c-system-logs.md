# 系统日志{#system-logs}

错误和审核日志可在安装路径 **[!UICONTROL Logs]** 的目录中找到。 这些日志对于审计仪表板系统和调试您可能遇到的任何问题非常有用。 日志使用服务器的当前日期存储在文件中：

Error logs: [!DNL thinclientweb-yyyy-mm-dd.log]

审核日志： [!DNL thinclientweb-audit-yyyy-mm-dd.log]
