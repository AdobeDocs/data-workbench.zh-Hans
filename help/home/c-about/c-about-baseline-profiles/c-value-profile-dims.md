# 值配置文件维{#value-profile-dimensions}

| 维度 | 类型 | 级别 | 描述 |
|---|---|---|---|
| 会话值 | 派生（基于“值”度量的MetricDim） | 访客 | 由业务价值模型定义的会话值。 |
| 价值细分 | 派生（基于“值”度量的MetricDim） | 访客 | 访客在其所有会话中生成的总值。 每个值段使用的默认增量为$50，但可以更改。 |
