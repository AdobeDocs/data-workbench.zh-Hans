# 值配置文件维度{#value-profile-dimensions}

| 维度 | 类型 | 级别 | 描述 |
|---|---|---|---|
| 会话值 | 派生（基于值量度的MetricDim） | 访客 | 由商业价值模型定义的会话值。 |
| 值区段 | 派生（基于值量度的MetricDim） | 访客 | 访客在其所有会话中生成的总值。 每个值区段使用的默认增量为$50，但可以更改。 |
