# 应用选择{#applying-selections}

选择不会自动应用于功能板的数据结果。

1. 完成所需选择并单击 **[!UICONTROL Update]**。

   如果按 **[!UICONTROL Update]** 钮为橙色，则表示必须单击该按钮才能在仪表板的选择中应用一些更改。 此功能允许您在屏幕上进行多项选择，并勾勒出分析问题，而无需每次进行更改时都启动查询。

   ![](assets/selection_update.png)
