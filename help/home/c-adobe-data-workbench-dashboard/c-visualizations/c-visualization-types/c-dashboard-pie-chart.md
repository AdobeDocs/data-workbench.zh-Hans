# 圆形分析图{#pie-chart}

饼图可视化允许您显示有限数量维度的度量值的相对百分比。 饼图允许最多20个唯一数据元素一次显示在图表上。 饼图的每个楔块表示数据集中的一个维度及其百分比。 每个百分比是相对于饼图中显示的其他维度的度量值计算的，而饼图上显示的其他维度不一定是整个数据集。

可通过单击任何特定楔形在饼图中进行选择。
