# 饼状图{#pie-chart}

饼图可视化允许您显示有限数量的维度的量度值的相对百分比。 饼图允许在图表上同时显示多达20个唯一数据元素。 饼图的每个楔块表示数据集中的一个维度及其百分比。 每个百分比是相对于饼图中显示的其他维度的量度值计算的，这些维度不一定是整个数据集。

单击任何特定楔形即可在饼图中进行选择。
