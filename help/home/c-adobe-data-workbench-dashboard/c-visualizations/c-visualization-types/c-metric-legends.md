# 量度图例{#metric-legends}

{{eol}}

量度图例可视化允许您以表格格式显示量度总计。 第一列保存量度名称，而第二列保存量度值。 当所有行都无法同时显示在窗口中时，将显示一个滚动条。

无法在量度图例中进行选择，因为未显示任何特定维度值。
