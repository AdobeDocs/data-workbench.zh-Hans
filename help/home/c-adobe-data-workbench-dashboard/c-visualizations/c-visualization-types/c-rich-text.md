# 富文本{#rich-text}

富文本可视化允许您在功能板中嵌入基于文本的信息。 这对于提供描述、超链接或指导性信息非常有用。 利用富文本窗口顶部的文本格式工具栏，可处理窗口主体中的文本格式。

![](assets/rich_text.png)

要向富文本窗口添加文本，请将光标放在富文本窗口的正文中，然后开始键入。 要在富文本窗口中编辑文本，请使用该窗口顶部的文本格式工具栏。
