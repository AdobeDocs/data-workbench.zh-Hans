# 柱状图{#column-charts}

柱状图可视化允许您使用图表上的垂直柱表示的每个数据元素可视化度量和维度数据。 柱状图允许在图表上同时显示多达200个唯一数据元素。 维值沿x轴标记，度量增量沿y轴标记。 每个维值的列将用其精确度量值进行标记。
