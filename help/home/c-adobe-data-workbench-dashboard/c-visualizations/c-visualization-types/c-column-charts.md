# 柱状图{#column-charts}

柱状图可视化允许您使用图表上的垂直柱表示的每个数据元素可视化量度和维度数据。 柱状图允许在图表上同时显示多达200个唯一数据元素。 维度值沿x轴进行标记，而度量增量沿y轴进行标记。 每个维度值的列都会用其精确量度值进行标记。
