# 可视化图表类型{#visualization-types}

介绍了八种不同的可视化类型。

在选择可视化时，必须仔细考虑要可视化的维度和量度，以及如何最有效地显示数据。 例如，某些可视化对一次可显示的数据量具有限制。

有八种不同的可视化类型可用于可视化和分析数据：条形图、柱状图、折线图、饼图、表格、量度图例、散点图和富文本。
