# 删除可视化{#removing-visualizations}

您可以随时从仪表板画布中删除可视化。

这是使用可视 **[!UICONTROL X]** 化窗口右上角的图标完成的。 这将从功能板画布中删除它，但不会影响任何已保存版本的功能板，除非保存当前功能板。 关闭的可视化中存在的任何选择都将从仪表板的选择中删除。
