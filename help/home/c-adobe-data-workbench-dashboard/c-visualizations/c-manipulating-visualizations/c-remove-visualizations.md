# 删除可视化图表{#removing-visualizations}

您可以随时从功能板画布中删除可视化。

可使用可视化窗口右上角的&#x200B;**[!UICONTROL X]**&#x200B;图标完成此操作。 这将从功能板画布中删除它，但不会影响任何已保存的功能板版本，除非保存了当前功能板。 关闭的可视化中存在的任何选择都将从仪表板的选择中删除。
