# 调整大小可视化{#sizing-visualizations}

除了展开和折叠可视化外，您还可以调整可视化的大小。

只需使用可视化窗口底部的调整大小手柄即可。 通过将鼠标悬停在可视化窗口的底边上，您可以单击并拖动以使窗口在仪表板画布上变高或变短。 功能板中的任何其他可视化将自动重新定位以适应调整后的可视化的新大小。 可视化的宽度相对于浏览器的宽度是固定的，无法操作。
