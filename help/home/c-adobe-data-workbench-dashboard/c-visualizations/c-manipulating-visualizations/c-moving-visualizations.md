# 移动可视化图表{#moving-visualizations}

{{eol}}

在功能板画布中，可以轻松地移动和重新排列可视化图表。

要移动可视化，只需单击可视化的标题栏，拖动可视化，然后将其拖放到功能板画布上的其他位置即可。 功能板画布将通过在所需位置为可视化腾出空间来，帮助定位可视化。 现有可视化图表会自动上下移动，以便为移动的可视化图表腾出空间以避免重叠。
