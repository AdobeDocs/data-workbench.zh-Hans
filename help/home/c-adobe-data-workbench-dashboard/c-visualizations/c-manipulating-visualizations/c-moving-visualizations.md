# 移动可视化{#moving-visualizations}

可在仪表板画布中轻松移动和重新排列可视化。

要移动可视化，只需单击可视化的标题栏，拖动可视化，然后将其放在仪表板画布上的其他位置。 仪表板画布将通过为可视化留出空间到所需位置来帮助定位可视化。 现有可视化将自动上移或下移，为移动可视化留出空间以避免重叠。
