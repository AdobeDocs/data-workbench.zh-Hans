# 移动可视化图表{#moving-visualizations}

可在仪表板画布中轻松移动和重新排列可视化。

要移动可视化，只需单击可视化的标题栏，拖动可视化并将其放到仪表板画布上的其他位置。 仪表板画布将通过在所需位置为可视化留出空间来帮助定位可视化。 现有可视化将自动上移或下移，为移动的可视化腾出空间以避免重叠。
