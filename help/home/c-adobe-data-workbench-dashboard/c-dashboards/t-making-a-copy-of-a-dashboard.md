# 制作功能板副本{#making-a-copy-of-a-dashboard}

复制功能板的步骤。

1. 加载要复制的功能板。
1. 从功能板界面顶部的功能板菜单中选择&#x200B;**[!UICONTROL Save As…]**。
1. 出现&#x200B;**[!UICONTROL Save Dashboard As…]**&#x200B;窗口时，输入功能板副本的新名称。
