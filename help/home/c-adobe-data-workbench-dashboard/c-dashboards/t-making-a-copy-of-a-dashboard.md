# 制作仪表板副本{#making-a-copy-of-a-dashboard}

{{eol}}

复制功能板的步骤。

1. 加载要复制的功能板。
1. 选择 **[!UICONTROL Save As…]** 功能板界面顶部的功能板菜单中。
1. 当 **[!UICONTROL Save Dashboard As…]** 窗口，输入功能板副本的新名称。
