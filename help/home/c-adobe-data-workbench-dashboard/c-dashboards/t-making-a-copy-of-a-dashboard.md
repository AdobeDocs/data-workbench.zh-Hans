# 制作功能板副本{#making-a-copy-of-a-dashboard}

复制仪表板的步骤。

1. 加载要制作副本的仪表板。
1. 从仪表板接口顶部的仪表板菜单中选择&#x200B;**[!UICONTROL Save As…]**。
1. 出现&#x200B;**[!UICONTROL Save Dashboard As…]**&#x200B;窗口时，输入仪表板副本的新名称。
