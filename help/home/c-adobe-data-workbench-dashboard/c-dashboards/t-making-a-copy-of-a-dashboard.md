# 制作功能板副本{#making-a-copy-of-a-dashboard}

复制功能板的步骤。

1. 加载要复制的功能板。
1. 从功能板界面顶部的功能板菜单中选择**[!UICONTROL Save As…]**。
1. 显示窗 **[!UICONTROL Save Dashboard As…]** 口时，输入功能板副本的新名称。
