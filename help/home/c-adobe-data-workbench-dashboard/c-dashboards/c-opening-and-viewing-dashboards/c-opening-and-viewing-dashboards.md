# 打开和查看功能板{#opening-and-viewing-dashboards}

仪表板可以通过以下三种方式之一打开：通过“快速访问”菜单、“仪表板浏览器”或使用“仪表板”超链接。 虽然仪表板一次只能在仪表板界面中打开一个，但您可以创建单独的浏览器选项卡或窗口，以同时视图多个仪表板。
