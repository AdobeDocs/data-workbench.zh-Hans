# 打开和查看功能板{#opening-and-viewing-dashboards}

可以通过以下三种方式之一打开功能板：通过快速访问菜单、功能板浏览器或使用功能板超链接。 虽然功能板界面中一次只能打开一个功能板，但您可以创建单独的浏览器选项卡或窗口，以同时查看多个功能板。
