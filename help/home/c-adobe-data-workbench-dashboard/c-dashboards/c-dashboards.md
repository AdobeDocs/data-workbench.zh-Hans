# 仪表板{#dashboards}

{{eol}}

创建功能板以可视化量度并提供具有数据的交互式分析功能。 通过单击功能板中的项目，您可以快速轻松地对数据进行分段，以从分析中获取信息。 可以创建无限数量的功能板，用于短期即时分析或长期功能板功能。 功能板可以保留为私有，也可以与其他功能板用户轻松共享。 您还可以从任何功能板导出数据，以供在其他工具(如Microsoft® Excel™)中使用。
