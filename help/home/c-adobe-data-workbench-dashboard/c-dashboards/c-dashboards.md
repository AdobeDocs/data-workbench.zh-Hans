# 功能板{#dashboards}

创建仪表板以可视化指标并提供交互式数据分析功能和数据。 通过单击功能板中的项目，您可以快速轻松地细分数据以从分析中派生信息。 可以创建无限数量的仪表板，用于短期即时分析或长期仪表板功能。 仪表板可以保持为私有，也可以与其他仪表板用户轻松共享。 您还可以从任何功能板导出数据，以用于Microsoft® Excel™等其他工具。
