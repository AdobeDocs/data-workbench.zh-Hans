# 功能{#features}

Adobe Data Workbench控制面板提供了这些功能。

* 对Adobe Data Workbench数据集进行实时分析。
* 使用简单的鼠标驱动界面对数据进行向下钻取和切片。
* 创建并保存多个分析仪表板。
* 与他人共享分析仪表板。
* 访问其他用户创建的仪表板。
* 导出数据以用于其他应用程序，如Microsoft® Excel®。

本指南提供了有关Adobe Data Workbench控制面板提供的功能和用户界面功能的信息。 由于您可以完全自定义仪表板的实施，因此您的功能和特性可能与本指南中介绍的内容有所不同。
