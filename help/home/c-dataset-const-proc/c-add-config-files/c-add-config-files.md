# 其他配置文件{#additional-configuration-files}

有关数据集配置文件或其继承配置文件的 Dataset 目录中包含的其他一些配置文件的信息。

尽管定义数据集配置文件并不总是需要这些文件，但您可以对这些文件进行编辑以指定其他参数来处理数据。
