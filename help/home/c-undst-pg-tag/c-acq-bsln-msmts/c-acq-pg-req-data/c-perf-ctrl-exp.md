# 执行受控实验{#performing-controlled-experiments}

传感器支持受控网站试验，允许您对访客群体的随机子集进行受控实验，同时将对当前操作的中断降至最低。

受控实验允许对提高网站财务绩效、改善客户体验或影响客户行为的假设进行试验。 可以在受控百分比的随机选择的网站访客上测试营销计划、内容个性化计划、流程以及创意或内容概念假设，然后在数据工作台中进行分析以确定这些假设的优点。
