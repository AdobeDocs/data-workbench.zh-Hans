# 使用元素点层{#working-with-element-point-layers}

在 Data Workbench 中，元素点层在地球上针对维度的每个元素显示一个点。

通过计算要绘制其元素的维度上的指定量度，可确定在地球上显示的点的大小。因此，较大的量度值对应地球上较大的点。
