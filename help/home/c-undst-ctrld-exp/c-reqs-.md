# 有哪些要求？{#what-are-the-requirements}

要使用“站点”在您的网站上进行受控试验，您必须满足以下要求：

* [!DNL Sensor] 必须在支持您正在测试的网站内容的每台Web或应用程序服务器上正确安装和运行。

   >[!NOTE]
   >
   >在J2EE服务器（JBoss、Tomcat、WebLogic和WebSphere）上运行的传感器不支持受控试验。

* 您必须有一个将内容推送到所有Web或应用程序服务器（如内容管理系统）的过程。
