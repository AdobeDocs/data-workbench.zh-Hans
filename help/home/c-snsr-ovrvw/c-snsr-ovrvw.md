# 传感器概述{#sensor-overview}

{{eol}}

传感器是一款轻量级、影响小的数据获取软件应用程序，可靠、安全、连续地从Web、应用程序和数据收集服务器收集事件数据以传输到任何Data Workbench Server。

[!DNL Sensor’s] 该工具可确保始终如一地准确测量Internet渠道中发生的事件。
