# 传感器概述{#sensor-overview}

传感器是一款轻量、低影响的数据获取软件应用程序，可靠、安全、连续地从Web、应用程序和事件收集服务器收集数据并传输到任何数据工作台服务器。

[!DNL Sensor’s] 该工具可确保对Internet渠道中发生的事件进行一致、准确的测量。
